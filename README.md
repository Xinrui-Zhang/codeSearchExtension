# Code Search 服务

本项目提供了项目工程代码导入，根据注释等信息检索相关代码的功能，分为 VScode 插件与桌面程序两部分，各部分运行方式见各对应文件夹下的 README。

- Extension 文件夹下为代码检索插件的源码
- DataInputApp 文件夹下为处理、导入检索数据的桌面程序源码
